**GIT Comands 1**
1) "git init" - Ввести команду надо в директории, чтобы организовать гит репозиторий;
2) "git add <название файла>" - начать отслеживание файла;

3) "git commit"(после ввода требуется ввести коментарий) - внести состояние текущего файла в базу данных;
4) "git commit -am <'Комментарий'>" - Команда которая выполняет сразу коммит, адд и коментарий;

5) "git log" - Посмотреть список коммитов, изменений;
6) "git log --oneline" - Более компактный вариант git log;

7) "git --hard reset <Хеш коммита на который нужно откатиться>" - Откатиться до указанной версии.
Хеш нужной версии можно посмотреть в логах.

8) "git clone <ссылка на директорию>" - ссылку можно взять из дириктории гита (HTTPS). 
Эта команда копирует директорию с гита на компьютер.
9) "git pull origin main" - Вводить внутри скопированной директории. Обновляет директорию на компьютере.

10) "git push" - Отправить из локальной(скопированной) директории изменения на гитхаб.
